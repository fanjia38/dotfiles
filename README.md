:wrench:  my .dotfiles

## install

```sh
# brew install
sh install_brew.sh

cd ~
git clone https://github.com/fanjia38/dotfiles.git
cd dotfiles

# set up
sh install.sh
```

## vscode settings

1. install command line tools
  ```
  Shift + Command + P
  ```
2. select `Shell Command: Install 'code' command in PATH command.`
3. run
  ```sh
  cd ~/dotfiles/vscode
  sh install.sh
  ```
